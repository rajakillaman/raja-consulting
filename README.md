# Velvet Vow Systems

One-page Next.js storefront for a one-time-payment digital template business
focused on wedding professionals.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind v4 base import + custom CSS

## Local development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`.

## Project shape

- `app/page.tsx`: storefront sections and product catalog data
- `app/globals.css`: full visual system and component styling
- `app/layout.tsx`: fonts + metadata
